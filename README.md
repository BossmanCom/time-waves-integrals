# Time Waves — Classic Improper Integrals

Interactive visualization of three celebrated integrals over the real line:

- **Gaussian**: ∫ e^{-x²} dx = √π
- **Lorentzian**: ∫ 1/(x² + 1) dx = π
- **Dirichlet (sinc)**: ∫ sin(x)/x dx = π

Watch the curves and the running integral accumulate toward their famous closed-form values.
